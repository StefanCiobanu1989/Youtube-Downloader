How to Run:

1.) Fist download the released file or compile your own.<br>
2.) Once downloaded unzip the contents into a directory called "YoutubeDownloader".<br>
3.) Open the app. The first time the app executes it will add ffmpeg to the system path.<br>
4.) Once you get the message "Folder added to system PATH." click ok and restart the app.<br>
5.) When running the app again it should take you to youtube's page.<br>
6.) Before being able to download anything you must first open Firefox and authenticate on youtube in Firefox. This is required because firefox youtube cookies are being used by the app.<br>
7.) After you have authenticated in Firefox close Firefox and start YoutubeDownloader.exe<br>
8.) While inside YoutubeDownloader navigate to a video of your choosing and wait for to "Download" button to show up on the right top corner of the youtube page. <img width="1046" height="71" alt="image" src="https://github.com/user-attachments/assets/9d9a65d3-e358-4249-948b-71b0404b1b21" />
<br>
9.) Click on the download button and chose the video source you want to download or download it as audio (mp3).<br><br>

To build this project you need visual studio 2022
